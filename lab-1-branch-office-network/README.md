# Lab 1: Branch Office Network Turn-Up

## Objective
Simulate a branch office network and perform a Day 1 network turn-up, including IP configuration and connectivity testing.

## Topology
- 2 PCs
- 1 Switch
- 1 Router
- LAN subnet: 192.168.1.0/24

## Tools Used
- Cisco Packet Tracer

## Steps Performed
1. Created network topology in Packet Tracer
2. Assigned IP addresses to PCs
3. Configured router interface
4. Verified connectivity using ping
5. Troubleshot misconfigured IP settings

## Troubleshooting
- Issue: PC could not reach the default gateway
- Resolution: Corrected subnet mask and gateway IP

## Outcome
Successfully verified end-to-end connectivity and documented Day-1 turn-up process.
